# 🎮 GAME ROOM

ブラウザで遊べるゲーム集です。

## 🎮 ゲーム一覧

### 💣 MINE SWEEPER
地雷を避けてフィールドを開拓する論理パズル。

### ♠ CARTAE
スペードとハートを6×6グリッドに配置するトランプ牌の論理パズル。

### # SUDOKU

9×9または6×6のグリッドに1〜9（または1〜6）の数字を配置する論理パズル。
各行・列・ブロックに同じ数字が重複しないように埋めよ。
